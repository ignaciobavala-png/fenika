# Fenika Studio — Carta de Presentación

Sitio web estático de presentación para **Fenika Studio**, agencia de comunicación estratégica y PR con foco en marcas, emprendedores y proyectos con propósito en LATAM.

## Contenido

Página única (single-page) con las siguientes secciones:

- **Hero** — Identidad y propuesta central
- **Por qué Fenika** — Propuesta de valor y diferenciales
- **Servicios** — 7 servicios de comunicación y PR
- **Metodología** — Proceso de trabajo en 5 pasos
- **Para quién** — Públicos objetivo
- **Founder** — Micaela Vitale Blisniuk, perfil y experiencia
- **Contacto** — Email, Instagram y WhatsApp

## Stack

HTML + CSS + JavaScript vanilla. Sin frameworks ni dependencias. Un solo archivo `index.html` autocontenido.

**Tipografías:** Epilogue · Manrope · Space Grotesk (Google Fonts)
